Bossfight
You can either run the bossfight in the godot editor and export it however you like, or you could also use the itch.io link I made https://omeletted.itch.io/bossfight. If you are opening it in the Godot Editor, you can either run it in there, or you can export it to whatever you like.

I coded this in Godot 4.3, and learned a lot about how shaders, global variables and tweens work. This was really fun, and this is the first game I have ever finished in godot, and I am really proud of it.
